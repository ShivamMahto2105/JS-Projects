# Simple Date & Time Display

This is a simple web project that displays the current date and time dynamically on a clean, centered interface. The time updates every second, providing a real-time clock experience.

## Preview

![Preview of Simple Date & Time](https://github.com/ShivamMahto2105/JS-Projects/blob/main/1%20Digital%20Clock/Preview.png)  
*Screenshot of the date and time display*

## Features

- Real-time digital clock
- Current date display
- Stylish and responsive design using CSS
- Easy to integrate into other projects

## Demo

Open `index.html` in your browser to see the live date and time.

## Installation

1. Clone or download the repository.
2. Make sure `index.html`, `style.css`, and `script.js` are in the same folder.
3. Open `index.html` in any modern web browser.

## Usage

- The date and time update automatically every second.
- You can customize the styling in `style.css` to match your preferences.


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
