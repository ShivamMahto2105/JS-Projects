# ⏱ Simple Stopwatch / Timer App

This is a simple web project that displays a functional stopwatch with start, stop, reset, and time-saving features. The timer updates dynamically and provides a clean, interactive user experience.

## Features

- Start the timer
- Stop the timer
- Reset the timer to zero
- Save multiple time records
- Clear saved time records
- Clean and responsive UI
- Built with pure JavaScript (Vanilla JS)

---

## Demo

Open `index.html` in your browser to see the live stopwatch in action.

---

## Installation

1. Clone or download the repository.
2. Make sure `index.html`, `style.css`, and `script.js` are in the same folder.
3. Open `index.html` in any modern web browser.

---

## Usage

- Click **Start** to begin the timer.
- Click **Stop** to pause the timer.
- Click **Reset** to set the timer back to zero.
- Click **Get Time** to save the current time.
- Click **Clear Time** to remove all saved times.
- You can customize the styling in `style.css`.
